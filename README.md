# Desafio DATA1 — Modelagem Preditiva de Vendas (Big Mart)

## Sobre o projeto
Este projeto tem como objetivo prever o volume de vendas (`OutletSales`) de produtos 
em diferentes lojas da rede Big Mart, utilizando técnicas de Ciência de Dados e 
Machine Learning.

## Tecnologias utilizadas
- Python
- Pandas e NumPy
- Matplotlib e Seaborn
- Scikit-Learn

## Estrutura do projeto
- `notebook.ipynb`: notebook com todo o processo de análise, limpeza, 
feature engineering e modelagem.

## Como executar
1. Abra o notebook no Google Colab ou Jupyter
2. Faça upload do arquivo `Train-Set.csv` 
3. Execute as células em ordem, de cima para baixo

## Resultados
O modelo final (Regressão Linear) obteve R² de aproximadamente 0.58 na base de 
validação. Mais detalhes sobre o processo e as decisões técnicas estão documentados 
diretamente no notebook.
